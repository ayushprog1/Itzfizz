# Itzfizz Web Development Assignment
A scroll-driven hero section built with React, GSAP, and Tailwind CSS.

## 🚀 Tech Stack
- **React (Vite)**
- **GSAP & ScrollTrigger** (For the car movement and text reveal)
- **Tailwind CSS v4** (For modern, utility-first styling)

## ✨ Key Features
- **Magic Text Reveal:** Uses a layered approach where a green trail "uncovers" white text over a hidden background.
- **Scroll Pinning:** The section stays fixed in the viewport while the animation plays out.
- **Responsive Stats:** Information cards that stagger into view as the user scrolls.

## 🛠️ Installation
1. `npm install`
2. `npm run dev`
